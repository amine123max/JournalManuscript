# Copernicus Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://publications.copernicus.org/for_authors/manuscript_preparation.html
- Package URL: https://publications.copernicus.org/Copernicus_LaTeX_Package.zip

## Standard Entry Points

- Archive: `assets/official-templates/copernicus/Copernicus_LaTeX_Package.zip`
- Package root: `assets/official-templates/copernicus/template-package`
- Selected template or class: `template-package/template.tex`

## Primary Files

- `assets/official-templates/copernicus/template-package/copernicus.cls`
- `assets/official-templates/copernicus/template-package/template.tex`
