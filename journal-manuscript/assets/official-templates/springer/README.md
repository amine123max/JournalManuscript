# Springer Nature Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://www.springernature.com/gp/authors/campaigns/latex-author-support
- Package URL: https://cms-resources.apps.public.k8s.springernature.io/springer-cms/rest/v1/content/18782940/data/v12

## Standard Entry Points

- Archive: `assets/official-templates/springer/springer-nature-template.zip`
- Package root: `assets/official-templates/springer/template-package`
- Selected template or class: `template-package/sn-article-template/sn-article.tex`

## Primary Files

- `assets/official-templates/springer/template-package/sn-article-template/sn-article.tex`
