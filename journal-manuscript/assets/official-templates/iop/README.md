# IOP Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://publishingsupport.iopscience.iop.org/questions/latex-template/
- Package URL: https://publishingsupport.iopscience.iop.org/wp-content/uploads/2025/07/ioplatextemplate.zip

## Standard Entry Points

- Archive: `assets/official-templates/iop/ioplatextemplate.zip`
- Package root: `assets/official-templates/iop/template-package`
- Selected template or class: `template-package/iopjournal-template.tex`

## Primary Files

- `assets/official-templates/iop/template-package/iopjournal.cls`
- `assets/official-templates/iop/template-package/iopjournal-template.tex`
- `assets/official-templates/iop/template-package/iopjournal-guidelines.pdf`
