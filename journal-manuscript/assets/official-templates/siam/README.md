# SIAM Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://www.siam.org/publications/journals/authors/
- Package URL: https://tug.ctan.org/macros/latex/contrib/siam/siamltex.zip

## Standard Entry Points

- Archive: `assets/official-templates/siam/siamltex.zip`
- Package root: `assets/official-templates/siam/template-package`
- Selected template or class: `template-package/lexample.tex`

## Primary Files

- `assets/official-templates/siam/template-package/siamltex.cls`
- `assets/official-templates/siam/template-package/lexample.tex`
- `assets/official-templates/siam/template-package/siam.bst`
