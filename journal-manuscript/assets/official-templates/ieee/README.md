# IEEE Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://journals.ieeeauthorcenter.ieee.org/create-your-ieee-journal-article/authoring-tools-and-templates/
- Package URL: https://mirrors.ctan.org/macros/latex/contrib/IEEEtran.zip

## Standard Entry Points

- Archive: `assets/official-templates/ieee/IEEEtran.zip`
- Package root: `assets/official-templates/ieee/template-package`
- Selected template or class: `template-package/bare_jrnl.tex`

## Primary Files

- `assets/official-templates/ieee/template-package/IEEEtran.cls`
- `assets/official-templates/ieee/template-package/bare_jrnl.tex`
- `assets/official-templates/ieee/template-package/bare_jrnl_compsoc.tex`
