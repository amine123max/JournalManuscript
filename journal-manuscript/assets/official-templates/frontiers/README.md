# Frontiers Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://www.frontiersin.org/design/zip/Frontiers_LaTeX_Templates.zip
- Package URL: https://www.frontiersin.org/design/zip/Frontiers_LaTeX_Templates.zip

## Standard Entry Points

- Archive: `assets/official-templates/frontiers/Frontiers_LaTeX_Templates.zip`
- Package root: `assets/official-templates/frontiers/template-package`
- Selected template or class: `template-package/frontiers.tex`

## Primary Files

- `assets/official-templates/frontiers/template-package/frontiers.tex`
- `assets/official-templates/frontiers/template-package/FrontiersinHarvard.cls`
- `assets/official-templates/frontiers/template-package/FrontiersinVancouver.cls`
