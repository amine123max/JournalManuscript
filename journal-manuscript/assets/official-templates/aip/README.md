# AIP Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://publishing.aip.org/resources/researchers/author-instructions/
- Package URL: https://mirrors.ctan.org/macros/latex/contrib/revtex.zip

## Standard Entry Points

- Archive: `assets/official-templates/aip/revtex.zip`
- Package root: `assets/official-templates/aip/template-package`
- Selected template or class: `template-package/revtex/revtex4-2.cls`

## Primary Files

- `assets/official-templates/aip/template-package/revtex/revtex4-2.cls`
- `assets/official-templates/aip/template-package/revtex/sample/aip/aiptemplate.tex`
- `assets/official-templates/aip/template-package/revtex/aip/aipguide4-2.pdf`
