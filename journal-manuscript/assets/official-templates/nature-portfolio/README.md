# Nature Portfolio Shared Family Assets

This directory follows the standard local family-template structure used by this repository, but it is a shared-root family rather than an independently downloaded package family.

- `template-package/` points to the shared Springer Nature package root used by this family
- `README.md` explains the shared-root boundary
- `metadata.json` records the machine-readable source and sharing relationship
- `shared-family-policy.md` explains what may be reused and what still needs title-specific confirmation

## Source Basis

- Official source: https://www.nature.com/nature-portfolio/for-authors/formatting-guide
- Shared package source family: `springer`

## Standard Entry Points

- Package root: `assets/official-templates/nature-portfolio/template-package`
- Shared root origin: `assets/official-templates/springer/template-package`

## Shared Family Policy

See `shared-family-policy.md`.
