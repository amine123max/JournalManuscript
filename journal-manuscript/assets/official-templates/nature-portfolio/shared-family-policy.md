# Nature Portfolio Shared Family Policy

## Can This Family Template Be Reused?

Yes, but through the shared Springer Nature package root rather than an independently downloaded Nature Portfolio family archive.

## What May Be Shared

- the shared `template-package/` root
- the Springer Nature `sn-jnl` workflow reused for Nature Portfolio titles in this repository
- the family-level baseline for front matter, section flow, and bibliography options

## What Still Needs Journal-Level Confirmation

Nature Portfolio titles still need title-specific confirmation for style option, reporting requirements, figure density, and metadata expectations.

## Safe Rule

Use the shared Springer Nature package root as the Nature Portfolio family baseline, then tighten the manuscript against the live target-title guidance before calling anything submission-ready.
