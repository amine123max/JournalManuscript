# ACM Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://authors.acm.org/proceedings/production-information/preparing-your-article-with-latex
- Package URL: https://mirrors.ctan.org/macros/latex/contrib/acmart.zip

## Standard Entry Points

- Archive: `assets/official-templates/acm/acmart.zip`
- Package root: `assets/official-templates/acm/template-package`
- Selected template or class: `template-package/acmart.cls`

## Primary Files

- `assets/official-templates/acm/template-package/acmart.cls`
- `assets/official-templates/acm/template-package/acmart/acmguide.pdf`
