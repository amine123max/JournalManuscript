# Wiley Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://authorservices.wiley.com/author-resources/Journal-Authors/Prepare/LaTeX/index.html
- Package URL: https://authors.wiley.com/asset/WileyDesign.zip

## Standard Entry Points

- Archive: `assets/official-templates/wiley/WileyDesign.zip`
- Package root: `assets/official-templates/wiley/template-package`
- Selected template or class: `template-package/Optimal-Design-layout/Optimal-Design-layout.tex`
