# ACS Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://pubs.acs.org/page/4authors/submission/tex.html
- Package URL: https://mirrors.ctan.org/macros/latex/contrib/achemso.zip

## Standard Entry Points

- Archive: `assets/official-templates/acs/achemso.zip`
- Package root: `assets/official-templates/acs/template-package`
- Selected template or class: `template-package/achemso.cls`

## Primary Files

- `assets/official-templates/acs/template-package/achemso.cls`
- `assets/official-templates/acs/template-package/achemso/achemso-demo.tex`
- `assets/official-templates/acs/template-package/achemso/achemso.pdf`
