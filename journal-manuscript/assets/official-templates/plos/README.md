# PLOS Official Template Assets

This directory follows the standard local family-template structure used by this repository:

- archive zip at the family root
- `template-package/` as the local entry root
- `README.md` for human guidance
- `metadata.json` for machine-readable metadata
- `shared-family-policy.md` for the family reuse boundary

## Source Basis

- Official source: https://journals.plos.org/plosone/s/latex

## Standard Entry Points

- Archive: `assets/official-templates/plos/PLOS_Author_Template.zip`
- Package root: `assets/official-templates/plos/template-package`
- Selected template or class: `template-package/plos_latex_template.tex`

## Primary Files

- `assets/official-templates/plos/template-package/plos_latex_template.tex`
- `assets/official-templates/plos/template-package/plos_latex_template.pdf`
- `assets/official-templates/plos/template-package/plos2025.bst`
