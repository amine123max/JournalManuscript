# Skill Scripts

This directory keeps the two end-user scripts that are intended to remain part of the published package.

## Scripts

- `export_selective_skill_bundle.py`: exports a slim skill bundle that keeps the selected family library, the required template assets, and the installable `journal-manuscript/` skill package.
- `scaffold_family_manuscript.py`: creates a `paper/` starter workspace from a supported family template baseline, including `main.tex`, `references.bib`, `CAPTION_BANK.md`, and manuscript subfolders.

## Typical Commands

```powershell
python scripts/export_selective_skill_bundle.py --family ieee --archive
python scripts/export_selective_skill_bundle.py --family elsevier --archive
python scripts/scaffold_family_manuscript.py --family ieee --output-dir C:\work\my-paper
python scripts/scaffold_family_manuscript.py --family frontiers --output-dir C:\work\my-frontiers-paper
python scripts/scaffold_family_manuscript.py --family oxford --output-dir C:\work\my-oxford-paper
```

## Output Contract

- Slim bundles generated by `export_selective_skill_bundle.py` should contain the inner `journal-manuscript/` skill folder, the selected family profile directories, all journals that belong to those families, and only the referenced guide or template assets needed by those profiles.
- Family scaffolds generated by `scaffold_family_manuscript.py` should create a usable `paper/` workspace with `main.tex`, `references.bib`, `CAPTION_BANK.md`, `README_PAPER.md`, `figures/`, `tables/`, and `tables/generated/`.
- Family scaffolds may come either from a reusable family-wide template root or from the closest locally cached official journal template that now serves as the family baseline.
