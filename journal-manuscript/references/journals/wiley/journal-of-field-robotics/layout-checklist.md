# Journal of Field Robotics Layout Checklist

Use this checklist together with `verification.yaml` before calling the journal-specific layout submission-ready.

## Verification Boundary

- Official template package verified through `official_preview.tex/.pdf/.png`.
- Journal-specific layout compliance still requires the checklist below.

## Required Checks

- [ ] `class-file-and-template-mode` Class file and template mode: Confirm the manuscript uses the imported official Wiley USG class and the selected template mode consistent with the Wiley PDF design package.
- [ ] `frontmatter-and-wiley-metadata` Front matter and Wiley metadata: Confirm title, authors, correspondence, abstract, keywords, and publisher metadata follow the Wiley authoring guidance.
- [ ] `section-flow-and-journal-rules` Section flow and journal rules: Confirm the manuscript section hierarchy and journal-level expectations match the target Wiley guide.
- [ ] `figure-table-and-graphics-rules` Figure, table, and graphics rules: Confirm figure handling, table style, and graphics preparation remain consistent with the Wiley authoring package and the target journal guidance.
- [ ] `references-and-bibliography-style` References and bibliography style: Confirm the selected Wiley bibliography mode matches the target journal guidance.
- [ ] `journal-specific-note` Journal-specific note: Verify title page structure, appendix handling, and reference style before calling the layout submission-ready.
