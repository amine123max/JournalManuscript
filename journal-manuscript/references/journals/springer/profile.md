## Refined Format Baseline

- Primary LaTeX anchor: a journal-provided Springer class such as `sn-jnl`, or a title-specific package
- Typical authoring layout: single-column manuscript package with journal-controlled bibliography options
- Front matter: title, authors, affiliations, abstract, keywords, and journal metadata
- References: journal-specific numeric or author-year mode
- Figures and tables: conservative single-column author-manuscript layout unless the journal package defines something else
- Common special items: theorem environments, appendices, supplementary material, and funding or data statements

## Check Before Editing

1. Exact class file and option set
2. Front matter structure
3. Abstract and keyword requirements
4. Figure and table placement rules
5. Citation and bibliography mode
6. Required end sections and declarations
7. Compile toolchain
