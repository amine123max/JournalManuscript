# ACM Family Layout Checklist

Use this checklist together with `verification.yaml` before calling the journal-specific layout submission-ready.

## Verification Boundary

- Official template family verified through `official_preview.tex/.pdf/.png`.
- Journal-specific layout compliance still requires the checklist below.

## Required Checks

- [ ] `class-file-and-submission-mode` Class file and submission mode: Confirm the manuscript uses the official ACM acmart class and the review-oriented submission mode required by ACM.
- [ ] `frontmatter-and-acm-metadata` Front matter and ACM metadata: Confirm title, abstract, CCS concepts, keywords, authors, affiliations, and ACM-specific metadata match the official authoring guidance.
- [ ] `section-flow-and-length` Section flow and manuscript length: Confirm the section hierarchy and overall manuscript length match the target ACM journal's expectations.
- [ ] `figure-table-and-accessibility-rules` Figure, table, and accessibility rules: Confirm figure and table treatment, caption style, and accessibility requirements such as figure descriptions follow ACM guidance.
- [ ] `references-and-bibliography-mode` References and bibliography mode: Confirm the citation and bibliography mode matches the official ACM journal workflow and the target venue requirements.
- [ ] `journal-specific-note` Journal-specific note: Use this family baseline only when the exact ACM target journal has not yet been fixed; once the venue is known, tighten the manuscript against that journal's live ACM guidance.
