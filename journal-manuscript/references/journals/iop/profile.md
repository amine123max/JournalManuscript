## Check Before Editing

1. Exact class file and option set
2. Front matter structure
3. Abstract and keyword requirements
4. Figure and table placement rules
5. Citation and bibliography mode
6. Required end sections and declarations
7. Compile toolchain

## Safe Rule

Treat this file as a reusable family baseline with caution. The local IOP package can be used as the starting point for drafting and preview, but some IOP titles do not require authors to submit in that exact class format, so the target journal guidance still wins.
