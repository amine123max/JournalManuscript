## Refined Format Baseline

- Primary template anchor: Frontiers manuscript package and author guidelines
- Typical authoring layout: single-column manuscript with specialty metadata
- Front matter: title, authors, affiliations, correspondence, specialty section, abstract, and keywords
- Common mandatory items: data availability, conflict of interest, author contributions, funding, and ethics statements when relevant
- References: publisher-defined bibliography format
- Figures and tables: clean single-column manuscript layout with open-science reporting sections visible on page one or near the end matter

## Check Before Editing

1. Exact class file and option set
2. Front matter structure
3. Abstract and keyword requirements
4. Figure and table placement rules
5. Citation and bibliography mode
6. Required end sections and declarations
7. Compile toolchain

## Safe Rule

Treat this file as a reusable family baseline. The Frontiers package may be shared across sibling journals for drafting and preview, but specialty-section labeling, declarations, and any title-level reporting requirements still need journal-specific confirmation before final layout work.
