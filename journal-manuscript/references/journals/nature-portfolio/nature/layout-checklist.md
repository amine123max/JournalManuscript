# Nature Layout Checklist

Use this checklist together with `verification.yaml` before calling the journal-specific layout submission-ready.

## Verification Boundary

- Official template package verified through `official_preview.tex/.pdf/.png`.
- Journal-specific layout compliance still requires the checklist below.

## Required Checks

- [ ] `class-file-and-style-option` Class file and style option: Confirm the manuscript uses the imported official sn-jnl package with the style option most consistent with the target family or journal guidance.
- [ ] `frontmatter-and-journal-metadata` Front matter and journal metadata: Confirm title, authors, affiliations, abstract, keywords, and any journal-specific metadata follow the Springer Nature author guidance.
- [ ] `section-flow-and-family-rules` Section flow and family rules: Confirm the article structure and any family-specific reporting expectations match the target journal guidance.
- [ ] `figure-table-and-supplementary-rules` Figure, table, and supplementary rules: Confirm artwork, table treatment, and supplementary file handling remain consistent with the official Springer Nature guidance.
- [ ] `references-and-bibliography-style` References and bibliography style: Confirm the selected bibliography style option matches the target family or journal guidance.
- [ ] `journal-specific-note` Journal-specific note: Verify Nature-specific editorial expectations, figure density, and any live journal metadata before calling the layout submission-ready.
