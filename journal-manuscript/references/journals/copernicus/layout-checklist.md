# Copernicus Family Layout Checklist

Use this checklist together with `verification.yaml` before calling the journal-specific layout submission-ready.

## Verification Boundary

- Official template package verified through `official_preview.tex/.pdf/.png`.
- Journal-specific layout compliance still requires the checklist below.

## Required Checks

- [ ] `class-file-and-journal-abbreviation` Class file and journal abbreviation: Confirm the manuscript uses the official Copernicus class and the correct journal abbreviation for the target title.
- [ ] `frontmatter-and-geoscience-metadata` Front matter and geoscience metadata: Confirm title, authors, affiliations, abstract, running title, and discussion-paper metadata follow the target Copernicus journal guidance.
- [ ] `section-flow-and-open-review-rules` Section flow and open-review rules: Confirm the manuscript structure, appendix handling, and any open-review or discussion-paper workflow requirements match the target journal.
- [ ] `figure-table-and-declaration-rules` Figure, table, and declaration rules: Confirm artwork, table treatment, author contributions, competing interests, and acknowledgments follow the target journal guidance.
- [ ] `references-and-bibliography-style` References and bibliography style: Confirm whether the target journal expects BibTeX with the Copernicus style or manual bibliography handling.
- [ ] `journal-specific-note` Journal-specific note: Use this family baseline only when the exact Copernicus target journal has not yet been fixed; once the venue is known, tighten the manuscript against that journal's live guidance.
