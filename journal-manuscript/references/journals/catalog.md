## Expansion Rule

- Keep family baselines at `references/journals/<family>/`.
- Store each child journal as `references/journals/<family>/<journal>/`.
- Keep folder names lowercase and hyphenated.
- Add or refine `profile.md` when an official class file or author guide is available.
