# Applied Physics Letters Layout Checklist

Use this checklist together with `verification.yaml` before calling the journal-specific layout submission-ready.

## Verification Boundary

- Official template family verified through `official_preview.tex/.pdf/.png`.
- Journal-specific layout compliance still requires the checklist below.

## Required Checks

- [ ] `class-file-and-journal-option` Class file and journal option: Confirm the manuscript uses the official REVTeX class with the correct AIP journal option and review format.
- [ ] `frontmatter-and-author-block` Front matter and author block: Confirm title, authors, affiliations, abstract, and any AIP-specific metadata follow the official author instructions.
- [ ] `section-flow-and-letter-constraints` Section flow and letter constraints: Confirm the article structure and any letter-format constraints match the target AIP journal guidance.
- [ ] `figure-table-and-data-availability` Figure, table, and data availability rules: Confirm artwork preparation, table style, and data availability statements follow AIP guidance.
- [ ] `references-and-revtex-style` References and REVTeX style: Confirm the reference style and REVTeX bibliography handling remain consistent with AIP instructions.
- [ ] `journal-specific-note` Journal-specific note: Verify APL letter-format constraints, rapid-communication expectations, and figure-density limits before calling the layout submission-ready.
