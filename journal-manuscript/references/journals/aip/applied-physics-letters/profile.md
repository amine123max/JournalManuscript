Use this profile when the target manuscript is intended for Applied Physics Letters.

- Parent family: AIP Family
- Start from: ../aip/profile.md


## Official Template Display

- Official guide page: https://publishing.aip.org/resources/researchers/author-instructions/
- Template basis: official AIP family class (`revtex4-2`) recorded in `verification.yaml`
- Selected display baseline: `revtex4-2.cls` with `apl` journal option
- Guide alignment: Verify APL letter-format constraints, rapid-communication expectations, and figure-density limits before calling the layout submission-ready.

## Journal Focus

- Rapid-turnaround physics letters journal

## Check Before Editing

1. Verify the actual class file, template package, or sample source used by the journal.
2. Confirm front matter requirements, abstract and keyword format, and author metadata.
3. Confirm figure and table caption placement, width rules, and supplementary material handling.
4. Confirm citation style, bibliography tooling, and appendix or declaration sections.
5. Journal-specific note: Verify AIP template mode, reference style, and figure placement constraints

## Safe Rule

Treat this file as a starter profile until the official template or author guide has been loaded and verified.
