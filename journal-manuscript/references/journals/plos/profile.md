## Check Before Editing

1. Exact class file and option set
2. Front matter structure
3. Abstract and keyword requirements
4. Figure and table placement rules
5. Citation and bibliography mode
6. Required end sections and declarations
7. Compile toolchain

## Safe Rule

Treat this file as a reusable family baseline. The local PLOS package may be shared across sibling journals for drafting and preview, but title-specific reporting and submission requirements still need confirmation before making high-risk layout changes.
